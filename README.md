# Youtube  
期末大作业，利用HTML+CSS完成一个仿Youtube的前端项目  
# 实现功能  
· 使用HTML和CSS制作了用户友好的界面，采用响应式设计适应不同屏幕尺寸。通过JavaScript集成动态元素，增强用户互动。
· 项目中设计了交互式侧边栏和高效导航系统，显著提升了用户体验和参与度
· 利用CSS的网格和弹性盒模型属性优化布局，确保在桌面、平板和手机上提供无缝的观看体验。
